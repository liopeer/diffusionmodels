Installation

If you don’t intend to make changes to the models I would use one of the General Usage, else I recommend going with one of the Development options.

General Usage

Conda

git clone https://github.com/liopeer/diffusionmodels.git
cd diffusionmodels
conda env create -f environment.yml
conda activate liotorch
pip install .
